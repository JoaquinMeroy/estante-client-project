# Estante — Inventory Management System

A fullstack inventory management system built for a drugstore client. Features a drag-and-drop interface that updates item positions in real-time to the database.

## ✨ Features
- Drag and drop inventory reordering
- Position changes persist to the database in real-time
- Clean, responsive UI
- User authentication via Google Sign-In

## 🛠️ Tech Stack
- **React** — component-based UI
- **Firebase** — real-time database and authentication
- **CSS** — custom styling

## 🚀 Getting Started

```bash
git clone https://github.com/JoaquinMeroy/estante-client-project.git
cd estante-client-project
npm install
npm run dev
```
